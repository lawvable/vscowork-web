# VSCowork (Web Version)

VSCowork is a pre-configured environment ready for AI-assisted document work. Try it now in your browser:

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/lawvable/vscowork-web?quickstart=1)

## What this does

Clicking the button above opens a [GitHub Codespace](https://github.com/features/codespaces) (a cloud development environment hosted by GitHub) with the [Lawvable extension](https://marketplace.visualstudio.com/items?itemName=lawvable.minimal-agent-workspace) pre-installed. That's it. You can get a glimpse of what VSCowork offers in the VS Code desktop app... directly from your browser!
